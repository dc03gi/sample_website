Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
Table Table Include a table
tr table row Include a row of data
th table header make a table header
td table data include a table data cell

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong emphasized text <strong>technical sophistication</strong> technical sophistication
a anchor make hyperlink <a href="https://www.example.com/example"><An example></a> example
img image include an image <img src="images/kitten.jpg" alt="a kitten">

Document Tags

Tag Short Description Long Description
html html start of html doc Reprents the root of an HTML document
head html start of html doc Provides information about the document including its title and links to its scripts and style sheets
body html start of html doc Represents the content of an html document
title html start of html doc It is the title that shows in either the top of the browser or on the browser tab
meta html start of html doc Represents metadata that cannot be represented by other HTML meta-related elements, like base, link, script, style, or title
